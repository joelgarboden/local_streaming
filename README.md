# Local streaming

## Setup

### Docker

Local clone: `DNS=$HOSTNAME docker-compose up --build`

### OBS

`Stream Type`: `Custom Streaming Server`
`URL`: `rtmp://DOCKER_HOSTNAME/live`
`Stream Key`: `stream`

### Browser

<http://DOCKER_HOSTNAME:8080/>
